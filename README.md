# Mis Finanzas

Plataforma web de gestión financiera personal orientada al usuario argentino. 
Desarrollada como proyecto académico en UADE.

## Funcionalidades

- **Calculadora de cuotas** — Simulación de pagos con interés.
- **Divisor de gastos** — Liquidación grupal con algoritmo de compensación de saldos.
- **Suscripciones** — Control de gastos mensuales en servicios digitales (USD/ARS).
- **Promociones** — Cálculo de precio real con descuentos y topes de reintegro.
- **Conversor de divisas** — Cotizaciones en tiempo real (Oficial, Blue, MEP, CCL, Tarjeta).
- **Herramientas financieras** — Conversor TNA→TEA, simulador de plazo fijo y préstamos 
  por sistema francés con tabla de amortización.

## Stack

- HTML5 semántico
- CSS3 con variables y diseño mobile-first
- JavaScript vainilla (ES6+)
- Bootstrap 5.3 (utilidades de grid y componentes puntuales)
- FontAwesome 6 (iconografía)
- DolarAPI (cotizaciones en tiempo real)

## Características

- 100% client-side: sin backend, sin login, sin almacenamiento en servidores.
- Mobile-first con breakpoint en 768px.
- Paleta "Growth & Clarity" y tipografías Outfit / Inter.
